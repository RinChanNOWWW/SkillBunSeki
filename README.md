# SkillBunSeki

A GUI tool to generate wiki source codes for [BEMANIWIKI SDVXV SKILL ANALYZER page](https://bemaniwiki.com/index.php?SOUND%20VOLTEX%20VIVID%20WAVE/SKILL%20ANALYZER).

## Develop

This app is developed with electron and React. This prject is generated by the template: https://github.com/electron-react-boilerplate/electron-react-boilerplate

### Start in Dev Mode

```bash
yarn
yarn start
```

### Pack 

```bash
yarn package
```
